Para el funcionamiento de la práctica, el usuario deberá introducir las
dimensiones del tablero deseado así como el número de turnos a jugar.

Seguidamente tendrá que introducir las coordenadas de las células que 
quiere inicializar como vivas en el Turno 0 y su tipo.

Tipos de células:
Célula 1: 23/3
Célula 2: 245/368
Célula 3: 51/346
Célula x: 1/1

Info. importante: Si el usuario desea crear una célula del tipo 'x'
deberá introducir un '4' por terminal para ser reconocido.

Finalmente, una vez enseñado el último turno, el usuario podrá decidir
si desea iniciar el juego nuevamente, volviendose a repetir todo el 
proceso.
